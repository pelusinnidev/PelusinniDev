Pol's Asteroids

A modern take for the classics!

Game Python MongoDB

Overview

In my journey as a developer, I chose to learn Python through reimagining the iconic Asteroids game. This modern interpretation offers both single and multiplayer experiences, leveraging the latest technologies and MongoDB integration for cross-device score tracking and competitive tournaments.

Experience the nostalgia of a classic, enhanced with contemporary features that bring new life to this timeless arcade game.

Key Features

Multiplayer Mode

Challenge your friends in local multiplayer battles or compete online in tournaments.

Global Leaderboard

Track your progress and compete with players worldwide through MongoDB-powered leaderboards.

Modern Graphics

Enjoy smooth animations and polished visuals while maintaining the classic arcade feel.

Custom Controls

Personalize your gaming experience with fully customizable controls and settings.

Technical Stack

  • Python with Pygame for game engine
  • MongoDB for data persistence
  • Socket.io for real-time multiplayer
  • Custom pixel art and animations
