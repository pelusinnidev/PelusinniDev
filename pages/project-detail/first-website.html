Overview

My first venture into web development: a fast food ordering website built entirely by hand (with just a sprinkle of AI for the logo). Looking back at this project and comparing it to my current portfolio shows an incredible journey of growth and learning in web development.

What makes this project special is that every line of code was written manually, learning the fundamentals of web development through trial and error. From figuring out how to style a button to making the menu items look appetizing, each challenge was a stepping stone to becoming a better developer.

Key Features

Interactive Menu

Browse and customize your favorite fast food items.

Order System

Simple yet functional ordering process with basic JavaScript.

Hand-Crafted UI

Every style and animation coded from scratch.

The Learning Journey

This project represents more than just code - it's a testament to the learning process. From the excitement of getting my first button to work, to the frustration of debugging CSS layouts, every challenge taught me something valuable. The website might not be perfect, but it's a genuine reflection of where my coding journey began.

Fun fact: I spent three hours trying to center a div (a rite of passage for every web developer), and the celebration when I finally got it right was unforgettable!
