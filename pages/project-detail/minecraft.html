Overview

A custom Minecraft server with unique plugins, automated deployment using Docker, and a web interface for server management. Supports multiple game modes and custom events.

Optimized server deployment on Oracle Cloud infrastructure.

Dual Versions

Both vanilla and Forge modded server options available.

Community Focus

Designed for friends to play and connect together.

While there are no active servers running at the moment, we're always ready to spin up new ones based on community interest. Whether you're looking for vanilla gameplay or a modded experience, feel free to reach out. We're open to suggestions and ready to accommodate server requests!
